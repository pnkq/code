Software

I have written some fundamental tools for Vietnamese text processing which were widely used by the community. The toolkit includes a word segmentation tool, a part-of-speech tagger, a dependency parser, a named entity recognizer. These tools are all open source and published under the GNU GPL. Here are some links to the tools:
  • Vietnamese Language Processing Demo
  • vlp
  • vitk (old toolkit, written in Java)
  • vitk-ner (old toolkit, written in Scala)
Most of my programs are currently written in the Julia programming language. They will be made open soon.
